### 日历使用
如果不需要修改的话，直接把dist目录下的拿去就行了。
如果需要修改源码（在src目录下），请按照下面方法：
#### 安装依赖
```
npm install
```

####  启动项目
```
npm start
```
启动项目之后，在src里面所做的修改会自动同步到dist目录的
